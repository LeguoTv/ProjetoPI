Sobre o Projeto

Este projeto tem como objetivo criar uma aplicação que auxilia no gerenciamento financeiro pessoal. A aplicação permite o controle de receitas e despesas, criando relatórios detalhados e gráficos que ajudam o usuário a entender seu fluxo de caixa.

A tecnologia usada no desenvolvimento inclui linguagens como HTML, CSS, PHP e JavaScript, com o suporte de banco de dados MySQL para armazenar as informações financeiras.

Sobre os Alunos

O projeto foi desenvolvido por dois alunos do 2º Módulo do curso Técnico de Desenvolvimento de Sistemas (TDS). Durante a criação desta aplicação, os alunos tiveram a oportunidade de aplicar os conhecimentos adquiridos nas disciplinas de programação, desenvolvimento web e banco de dados, consolidando seu aprendizado em um projeto prático.

Esta experiência permitiu que ambos evoluíssem tanto tecnicamente quanto na capacidade de trabalhar em equipe, dividindo responsabilidades e colaborando para resolver problemas e implementar funcionalidades essenciais para o bom funcionamento da aplicação.
